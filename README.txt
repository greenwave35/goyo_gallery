전시 워크쓰루 패키지 (assets_new)

1) 웹(HTML) 버전 보기
- images/ 폴더와 같은 위치의 index.html 을 브라우저로 여세요.
- 작품 이미지를 누르면 크게 보기(좌/우 이동 가능).

2) PDF 버전
- '전시_워크쓰루.pdf' 파일을 바로 열면 됩니다.

구성
- index.html : 전시 동선(섹션 3개) + 작품 24점
- metadata.json : 제목/설명/정보 매칭 데이터
- images/ : 작품 이미지(원본 webp)
- 전시_워크쓰루.pdf : 인쇄/공유용 PDF

※ 오프라인(파일)로도 열리도록 metadata.json을 별도 로드하지 않고, index.html 안에 직접 반영해 두었습니다.
